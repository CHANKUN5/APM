# 📦 Actividad 1.1: Modularización de la UI

## 🎯 Objetivo
Transformar el archivo único en un sistema de componentes atómicos y reutilizables.

## 🛠️ Acciones Realizadas
1.  **Identificación de Patrones Visuales:** Se detectaron elementos repetitivos como botones, tarjetas de estadísticas y etiquetas.
2.  **Creación de Atómicos:** Implementación de `StatCard`, `Badge`, `Button` y `TextField` en la carpeta `src/components/`.
3.  **Estilos Puros:** Eliminación de estilos inline caóticos en favor de una gestión más limpia y estructurada.

## 🎨 Resultado
Una interfaz que se ensambla como piezas de LEGO, facilitando cambios visuales globales sin afectar la lógica.
