# 🛠️ Actividad 1.3: Abstracción Funcional (Utilities)

## 🎯 Objetivo
Eliminar la duplicación de lógica auxiliar y formateo.

## 🛠️ Acciones Realizadas
1.  **Centralización:** Creación de `src/utils/formatters.js`.
2.  **Formateo de Moneda:** Implementación de `money()` usando `Intl.NumberFormat` para estandarizar la moneda PEN.
3.  **Gestión de Clases:** Creación de `classNames()` para manejar condicionales de CSS de forma elegante.

## 💎 Resultado
Código DRY (Don't Repeat Yourself) que asegura que la presentación de datos sea uniforme en toda la aplicación.
